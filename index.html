Wireless EV Charging System

Arduino-based prototype for wireless electric vehicle charging using Inductive Power Transfer (IPT). The system automatically detects vehicle positioning using an IR sensor array with 95% accuracy and manages the full charging sequence — from detection to relay control — with emergency shutdown capability. Achieved 85% power transfer efficiency through resonant circuit tuning.

  • Arduino
  • Embedded C++
  • IPT
  • IR Sensors
  • IoT

Smart Staircase Lighting System

Motion-activated staircase lighting system using IR sensors and Arduino. Lights activate sequentially when motion is detected and shut off automatically after a configurable timeout — no manual switching needed. Uses non-blocking millis() timing logic to avoid delays. Achieved 45% energy reduction compared to always-on lighting.

  • Arduino
  • Embedded C++
  • IR Sensors
  • Automation

Edge-Avoiding Autonomous Robot

Autonomous navigation robot using an ultrasonic sensor array and Arduino motor control. The firmware continuously reads sensor data, applies averaging and calibration filters to improve edge detection accuracy by 40%, and uses threshold-based decision logic to avoid obstacles and edges in real time.
